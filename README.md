# QuanLyChiTieu
# Giới thiệu:
- Ứng dụng quản lý chi tiêu giúp bạn có thể quản lý chi tiêu cá nhân, ứng dụng kết hợp ghi chép chi tiêu hằng ngày và các biểu đồ biểu diễn mức độc chi tiêu
# Điều kiện cài đặt:
- nodejs 18+
- firebase
